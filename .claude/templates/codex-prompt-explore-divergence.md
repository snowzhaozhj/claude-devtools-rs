# codex explore 方向分叉 prompt 模板

## 何时用

explore 阶段出现 ≥2 可行方案且选择影响数据模型/模块边界/IPC/持久化/async/性能/用户可见行为（触发点 #9）。

## prompt 骨架

```
背景：[一句话讲要解决什么问题]

Leading option：[方案 A 描述，1-3 句]
Rejected option(s)：[方案 B/C 描述，各 1-2 句]

我选 A 的理由：
1. [理由 1]
2. [理由 2]

选择影响：[数据模型 / 模块边界 / IPC / 持久化 / async / 性能 / 用户行为 中哪些]

我最担心的假设：[A 成立的前提条件，可能不成立的场景]

请你：
1. 对 A 做对抗质询——我的理由哪个最脆弱？什么条件下 A 会比 B 差？
2. A 有没有我没看到的隐性成本（迁移 / 测试 / 性能 / 维护）？
3. 你会选什么？为什么？（允许和我不同）

约束：中文，直接给观点，≤ 500 字
```
